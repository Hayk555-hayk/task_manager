Vue js можно установить через cdn, npm и cli
Создания новой сущности vue 
new Vue({
    el: '#app', // MAin html tag where all app will be rendered inside the mentioned tag
    data: {
        name: 'Hayk', // Interpolation in html {{ name }}
    }, 
    methods: {
        greet: function() {
            return 'good morning' + this.name; // interpolation can be span{{ greet() }}/span
        }
    }
});
Производим связку данных с атрибутами html
