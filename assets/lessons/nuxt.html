Nuxt js в себе уже имеет vueX vue-router и создает ssr(server side rendering) приложение
У nuxt своеобразный роутинг который происходит через названия папок 
pages
    about
        info.vue //about/info
    users
        _id
            index.vue //users/user_id/index
Пошаговое создание nuxt app 1)npm init -y 2) npm i nuxt --save 3)create script in package json dev: 'nuxt' тепер npm run dev запустит наш сервер
После всего этого создается директория pages где и будет происходить весь роутинг, создается index.vue файл, который будет home page для проекта
Для перехода по другим страницам нужно просто прописать router-link to="/" home page /router-link
Кроме router-link есть и nuxt-link который и обеспечит SSR
Создание nuxt app через npx - npx-create-nuxt-app app-name
Если мы пропишем npm built то после нужно прописать npm start чтобы сгенерированный проект начал работать
Для добавки плагина в nuxt js в папке plugins создается нужный файл а в nuxt.config.js в массиве plugins: [@/plugins/pluginName]
